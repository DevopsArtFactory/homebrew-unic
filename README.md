# homebrew-unic

Homebrew tap for [unic](https://github.com/DevopsArtFactory/unic) - A Go-based TUI tool for browsing and managing AWS resources in the terminal.

## Installation

```bash
brew tap DevopsArtFactory/unic
brew install unic
```

## Upgrade

```bash
brew upgrade unic
```

## Uninstallation

```bash
brew uninstall unic
```
